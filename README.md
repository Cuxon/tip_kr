# NotesApp

Простое веб-приложение для заметок на React + Vite.

## Возможности
- Добавление заметок
- Удаление заметок

## Используемые технологии
- React 18
- Vite
- @vitejs/plugin-react
- JavaScript (ES6+)
- HTML и CSS

## Структура проекта
```

src/
├── main.jsx     # точка входа
├── App.jsx      # главный компонент
└── NotesApp.jsx # список заметок

```

## Как запустить
1. Установить зависимости:
```

npm install

```
2. Запустить проект:
```

npm run dev

```
3. Открыть в браузере `http://localhost:5173/`
